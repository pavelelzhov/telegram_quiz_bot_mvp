# AGENTS.md

## Rules
- Всегда работать через patch-скрипты, не делать хаотичные ручные правки.
- После изменений прогонять py_compile для затронутых Python-файлов.
- Не коммитить .env, токены, базы данных и runtime-артефакты.
- Сохранять совместимость с Windows PowerShell.
- Все пользовательские инструкции и пояснения писать простым русским языком.
- Для новых функций сначала минимально рабочий вариант, потом улучшения.
