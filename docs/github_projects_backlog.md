## Epic 3 — Качество квиза и баланс

### [P1] Адаптивная сложность
**Labels:** `game-design`, `backend`, `p1`

**Описание**
Подстраивать сложность под чат по сигналам: timeout/skip/correct.

**Текущие правила (реализация MVP)**
- первые 3 вопроса идут как `easy` (мягкий вход);
- если средний сигнал чата `>= 0.45` → цель `hard`;
- если средний сигнал `<= -0.30` → цель `easy`;
- иначе цель `medium`.

Сигналы: `correct = +1`, `close = 0`, `wrong/timeout = -1`.

**AC**
- [x] Есть явные правила повышения/понижения сложности.
- [x] Есть ограничители, чтобы не было резких скачков.
- [x] Правила отражены в документации.

**Estimate:** 8h
